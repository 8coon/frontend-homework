- Напишите функцию `chess`, которая рисует ASCII-шахматрую доску размером N*N символов из звёздочек (в левом верхнем углу всегда стоит звёздочка)
- Напишите функцию `tree`, которая рисует ASCII-ёлочку высотой N символов из звёздочек
- Напишите функцию `fromHex`, которая может переводить числа из 16-ричной СС в двоичную
- Напишите функцию `fromRoman`, которая позволяет переводить римские числа в десятичные
- Напишите функцию `euclid`, которая позволяет находить НОД трёх натуральных чисел
- Напишите функцию `plain`, которая будет принимать на вход массив массивов и создавать из них один общий массив
- Напишите функцию `inverse`, которая будет принимать на вход массив и возвращать новый массив, элементы в котором переставлены в обратном порядке за исключением первого
- Напишите функцию `sortString`, которая будет сортировать буквы в словах по алфавиту, а потом получившиеся слова в предложении
- Напишите функцию `toUpperCase`, которая сделает заглавной первую букву каждого слова строки, а остальные буквы сделает строчными
- Напишите функцию `zip`, которая будет принимать на вход несколько объектов (любое количество), и возвращать единственный объект, содержащий все поля из всех объектов. Если одно и то же поле было в нескольких объектах, необходимо оставить значение, которое встретилось раньше
- Напишите функцию `removeAllDuplicateLetters`, которая будет удалять из строки символы, которые встречаются в ней больше одного раза
- Напишите функцию `removeDuplicateLettersBegin`, которая будет искать повторяющиеся символы в строках и удалять их, оставляя на месте только первую встречающуюся букву
- Напишите функцию `removeDuplicateLettersEnd`, которая будет искать повторяющиеся символы в строках и удалять их, оставляя на месте только последнюю встречающуюся букву
- Напишите функцию `intersection`, которая будет принимать на вход несколько массивов и находить их пересечение
- Напишите функцию `union`, которая будет принимать на вход несколько массивов и находить их объединение
